# SmartFramework
简单的MVC框架，定义了一系列注解：通过Controller注解定义Controller类，通过Inject注解实现依赖注入，通过Action注解定义Action方法。
通过一系列Helper类来初始化MVC框架：通过DispatcherServlet处理所有请求。
尚未实现AOP
